# JS_Automation
Nightwatch test automation


cd to the root of the project

1. Install npm 

In your terminal run npm install


2. npm init

In your Terminal run the command npm init this will create a package.json file.
Press Enter for all questions.
3. Now we will install nightwatch using npm

Run command in terminal — npm install nightwatch


4. Terminal you can run — node nightwatch -e chrome -a tagname
